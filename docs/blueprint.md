# **App Name**: GreenNote

## Core Features:

- Create New Note: Allow users to create a new note with a title and content.
- Edit Note: Enable users to modify the content of an existing note.
- View All Notes: Display a list of all notes created by the user.
- Delete Note: Provide functionality to remove unwanted notes.

## Style Guidelines:

- Primary color: Soft green (#B2D7B2) for a calming and focused environment.
- Background color: Off-white (#F5F5F5) for a clean and minimalist design.
- Accent color: Slightly darker green (#86C475) for interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable interface.
- Use a simple, single-column layout to reduce clutter and increase focus.
- Incorporate subtle animations for transitions, like fading in/out notes or sliding effects when deleting notes, to give the app a polished feel without being distracting.
